# Fynx Fullstack Web Platform 

A comprehensive collaborative platform featuring a modern **Angular** frontend and a secure **Java Spring Boot** backend.

## Key Features
- **AI-Powered Chat:** Integrated Chatbot service for enhanced user interaction.
- **Secure Authentication:** Robust user security using **JWT (JSON Web Tokens)** and Spring Security.
- **Dynamic Content:** Full CRUD operations for categories, comments, and skills.
- **Media Support:** Built-in file upload and image management system.
- **Email Service:** Automated email notifications and updates.

## Tech Stack
- **Frontend:** Angular (Standalone Components), TypeScript, RxJS.
- **Backend:** Java Spring Boot, Maven, JPA/Hibernate.
- **Security:** Spring Security, JWT.
- **Database:** H2/MySQL with custom SQL initialization.

## Structure
- `client/`: Angular frontend application.
- `server/`: Spring Boot REST API.
